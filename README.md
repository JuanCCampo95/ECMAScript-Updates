# ECMAScript-Updates
In this repository you will find several ECMAScript Updates, since ES6 to ES13. 

By the way, these are my personal notes about this updates. and I invite you to help me improve them by pulling requests.

![](https://i.imgur.com/NzBWRfa.png)

## ECMASCRIPT 6 (2015)
- Let & Const variables
- Arrow Functions
- Strings applying backticks (Template literals)
- Default params
- Rest Spread
- Objects literals
- Promises
- Clases
- Module
- Generator
- Set & Add

## ECMASCRIPT 7 (2016)
- Exponetials
- Array Includes

## ECMASCRIPT 8 (2017)
- Object Entries
- Object Values
- String Padding
- Trailing Commas
- Async Functions

## ECMASCRIPT 9 (2018)
- Regex
- Spread (update)
- Finally
- Async (update)

## ECMASCRIPT 10 (2019)
- Flat-Map
- Trim Start y End
- Try Catch
- From entries

## ECMASCRIPT 11 (2020)
- Optional Chaining
- Big Ints
- Nullish
- Promise allSettled & all
- Globalthis
- Match All
- Dynamic Imports with modules

## ECMASCRIPT 12 (2021)
- Numeric Separator 100_000
- Replace All
- Promises Any
- Private methods
