# ECMAScript-Updates
In this repository you will find several ECMAScript Updates, since ES6 to ES13. 

By the way, these are my personal notes about this updates. and I invite you to help me improve them by pulling requests.

![](https://i.imgur.com/NzBWRfa.png)

## ECMASCRIPT 6 (2015)
- Let & Const variables
- Arrow Functions
- Strings applying backticks (Template literals)
- Default params
- Rest Spread
- Objects literals
- Promises
- Clases
- Module
- Generator
- Set & Add

## ECMASCRIPT 7 (2016)
- Exponetials
- Array Includes

## ECMASCRIPT 8 (2017)
- Object Entries
- Object Values
- String Padding
- Trailing Commas
- Async Functions
